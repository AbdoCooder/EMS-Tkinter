# Employee Management System

Desktop employee management app with Tkinter UI, SQLite database, and XML import/export.

## Quick Start

```bash
pip install lxml
python main.py
```

## Features

- Add, edit, delete employees
- Search/filter by name or department
- Import/export XML with DTD validation

## Files

| File | Description |
|------|-------------|
| `main.py` | Application entry point |
| `db_manager.py` | SQLite database operations |
| `xml_manager.py` | XML import/export |
| `ui_main.py` | Main window UI |
| `ui_form.py` | Employee form dialog |
| `employes.dtd` | XML schema definition |
